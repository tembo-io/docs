**Tembo** is designed to support engineering teams by identifying and solving issues in production with specialized pull requests. Whether you're interested in improving database health, fixing slow queries, or navigating best practices, **Tembo** offers a straightforward way to get code fixes, fast. Our documentation walks you through quick setup, provides background on architecture and basic functionality, and describes key features that you'll want to leverage when optimizing database performance.

## Architecture

Tembo is the first autonomous AI agent that closes the gap between database monitoring and software engineering![](/tembo-overview.png)By continuously detecting and solving issues, Tembo can help you quickly burn down your existing issues, detect regressions, and solve emergent issues before they become a problem.
