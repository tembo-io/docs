The Jira integration allows Tembo to automatically detect and solve issues from your Jira workspace. Simply add the "tembo" label to any issue and Tembo will start working on it.

## Setup

1. First, install the Jira integration from the [Tembo integrations page](https://app.tembo.io/integrations)
2. Add the "tembo" label to any Jira issue and Tembo will automatically start working on it. That's it!
