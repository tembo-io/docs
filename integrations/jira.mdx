The Jira integration allows Tembo to automatically detect and solve issues from your Jira workspace. Simply add the "tembo" label to any issue and Tembo will start working on it.

## Setup

Just add the "tembo" label to any Jira issue and Tembo will automatically start working on it. That's it!
