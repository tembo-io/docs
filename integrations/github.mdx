icon: "github"
---

- Click into settings, and click the `Connect` button next to Github in the Integrations section.
- Select which code repositories that you would like help maintaining.

<Info>
  Providing access to the repo where you create database migrations is especially important.
</Info>

## Link instances and repos

Once you've authorized repos:

- Click into each Postgres instance and link it with one or more code repos.

<Info>
  We will only look to make PRs for database issues discovered in the repos that you have expressly
  linked to each Postgres instance.
</Info>
