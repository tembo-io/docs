---
title: 'Pull Requests'
description: 'How Tembo raises github PRs for you'
icon: 'code-pull-request'
---

Tembo will attempt to solve issues automously, submitting pull requests when it is feels confident that it's come up with a helpful PR.

From there, you can manage the workflow using normal Github procedures. If you merge the PR, it will auto-resolve the related Tembo issue.
